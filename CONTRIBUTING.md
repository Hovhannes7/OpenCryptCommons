# Contributing to OpenCryptCommons

Thank you for your interest in OpenCryptCommons.

## Current status

This repository is currently a pre-award placeholder. The initial public purpose of the repository is to document project intent, roadmap direction, and future open development.

Implementation code is not yet available.

## How to contribute at this stage

At this stage, useful contributions include:

- comments on scope and architecture
- references to relevant standards or open-source implementations
- feedback on threat modelling and interoperability
- suggestions for real-world integration targets
- identification of related prior art or adjacent projects

## Before opening an issue

Please check whether your suggestion is already reflected in the README or roadmap notes.

## Expected contribution areas in the first phase

- documentation
- roadmap review
- interoperability notes
- example use cases
- threat model feedback

## Code contributions

Code contributions will be invited once the initial implementation roadmap is published.

## Communication

Please use GitHub Issues for technical feedback and suggestions.
